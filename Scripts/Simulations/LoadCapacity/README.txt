!!MatlabAnsysConnection.svg, gives a schematic overview of how these scripts are connected. 

Download all the files in this folder and than follow these steps.

To use these files:
1. Update the directory adress of POST,PRE,EXPORT,VMS,Results files in the CODEIN.mac, CODEOUT.mac and EXPORT.mac file. 
2. Also update the directory adress of your ANSYS installation in the LAUNCHIN.BAT and LAUNCHOUT.BAT file. 
3. The input to run these files is a matrix (r) of the parameters of viable solutions which can be found using the check function in the Data class. 
4. This matrix (r) can be input into RUNMUL.m.
